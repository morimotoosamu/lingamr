# Causal order search via pwling

Causal order search via pwling

## Usage

``` r
search_causal_order_pwling(X, U, Uc, Vj)
```

## Arguments

- X:

  Data matrix

- U:

  Indices of all variables

- Uc:

  Indices of candidate variables

- Vj:

  Variable set based on prior knowledge

## Value

Index of the selected variable
