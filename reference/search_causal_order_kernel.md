# Causal order search via the kernel method

Causal order search via the kernel method

## Usage

``` r
search_causal_order_kernel(X, U, Uc, Vj)
```

## Arguments

- X:

  Data matrix

- U:

  All variables

- Uc:

  Candidate variables

- Vj:

  Prior knowledge set

## Value

Index of the selected variable
