# Generate a noise matrix with an independent seed per variable

Column k is generated by `noise_fn(n)` immediately after
`set.seed(seed + k - 1)`. Fixing the seed per variable ensures that the
same `seed` always yields the same noise columns.

## Usage

``` r
generate_noise_matrix(n, n_vars, seed, noise_fn)
```

## Arguments

- n:

  sample size

- n_vars:

  number of variables (columns)

- seed:

  base seed. Column k uses seed + k - 1

- noise_fn:

  noise generation function of the form `function(n)`

## Value

noise matrix (n x n_vars)
