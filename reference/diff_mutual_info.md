# Difference of mutual information

Difference of mutual information

## Usage

``` r
diff_mutual_info(xi_std, xj_std, ri_j, rj_i)
```

## Arguments

- xi_std:

  Standardized xi

- xj_std:

  Standardized xj

- ri_j:

  Residual of xi regressed on xj

- rj_i:

  Residual of xj regressed on xi

## Value

Difference of mutual information
